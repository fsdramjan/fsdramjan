**GenPBX** — A mobile SIP/VoIP client built solo. Flutter handles every screen; a custom native Kotlin layer (raw WebSocket SIP signaling, foreground service, native call-screen Activity) handles incoming calls reliably across foreground, background, and app-killed states. Push-to-talk runs on MQTT, push notifications on Firebase, multi-account storage on Hive.
